# 헤어스타일 참조 이미지

이 폴더에는 AI 합성에 사용될 헤어스타일 참조 이미지를 저장합니다.

## 필요한 파일

다음 이미지 파일들을 이 폴더에 추가해야 합니다:

- `style_1.jpg` - 스타일 1 참조 이미지 (예: 짧은 단발 머리)
- `style_2.jpg` - 스타일 2 참조 이미지 (예: 긴 웨이브 머리)
- `style_3.jpg` - 스타일 3 참조 이미지 (예: 중간 길이 생머리)

## 이미지 요구사항

- **파일 형식:** JPG, PNG
- **권장 크기:** 512x512 ~ 1024x1024 픽셀
- **내용:** 명확한 헤어스타일이 보이는 정면 또는 측면 사진
- **배경:** 단순한 배경 권장 (헤어스타일에 집중)

## 이미지 예시

### style_1.jpg (짧은 단발)
- 어깨 위 길이의 단발 머리
- 다크 브라운 컬러
- 깔끔한 커트

### style_2.jpg (긴 웨이브)
- 어깨 아래 긴 머리
- 블론드 컬러
- 자연스러운 웨이브

### style_3.jpg (중간 길이 생머리)
- 어깨 정도 길이
- 블랙 컬러
- 스트레이트 헤어

## 이미지 찾는 방법

1. **무료 이미지 사이트:**
   - Unsplash (https://unsplash.com)
   - Pexels (https://pexels.com)
   - Pixabay (https://pixabay.com)

2. **검색 키워드:**
   - "woman hairstyle portrait"
   - "bob cut hairstyle"
   - "long wavy hair"
   - "straight hair portrait"

3. **AI 생성 이미지:**
   - DALL-E, Midjourney 등으로 생성 가능

## 주의사항

⚠️ **저작권:** 상업적 이용 가능한 이미지만 사용하세요.
⚠️ **얼굴:** 얼굴이 명확히 보이고 헤어스타일이 잘 보이는 이미지를 선택하세요.
